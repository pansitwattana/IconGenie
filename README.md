**IconGenie** is a premium, client-side web application that instantly converts your images into professional web icons. It supports all major formats including Favicon, iOS Touch Icons, Android PWA icons, and more.

![IconGenie Preview](https://via.placeholder.com/800x400?text=IconGenie+Preview)

## ‚ú® Features

- **Premium UI**: Glassmorphism design with fluid animations and a modern aesthetic.
- **Privacy First**: All processing happens **in your browser**. No images are uploaded to any server.
- **Multi-Format Support**: Generates standard sizes for:
  - Favicons (16x16, 32x32)
  - Apple Touch Icons (180x180)
  - Android PWA Icons (192x192, 512x512)
  - App Store (1024x1024)
- **Instant Zip Download**: Download all generated icons in a single click.
- **Drag & Drop**: Intuitive upload interface.

## üöÄ How to Use

1. **Open the App**: Simply open `index.html` in your browser.
2. **Upload Image**: Drag and drop your logo (PNG, JPG, or WEBP). High resolution (1024x1024) recommended.
3. **Select Sizes**: Choose the icon sizes you need (standard set is selected by default).
4. **Generate**: Click "Start Conversion".
5. **Download**: Download individual icons or get them all in a ZIP file.

## üåê Deployment (GitHub Pages)

You can host this tool for free using GitHub Pages.

### Step 1: Create a Repository
1. Go to [GitHub.com](https://github.com) and create a new repository (e.g., `icon-genie`).
2. Make it **Public**.

### Step 2: Upload Files
1. Upload `index.html` to the root of your repository.
2. (Optional) Add this `README.md`.

### Step 3: Enable GitHub Pages
1. Go to your repository **Settings**.
2. Click on **Pages** in the left sidebar.
3. Under **Build and deployment** > **Source**, select **Deploy from a branch**.
4. Under **Branch**, select `main` (or `master`) and `/ (root)`.
5. Click **Save**.

Wait a minute or two, and your site will be live at `https://<your-username>.github.io/icon-genie/`!
